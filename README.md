# 🎮 CodeCraft - TCC

CodeCraft é um jogo educacional desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de ensinar lógica de programação de forma interativa e prática.

---

## 📌 Funcionalidades

- ✅ Ensino de lógica de programação por meio de fases
- ✅ Sistema progressivo de dificuldade
- ✅ Mecânicas baseadas em:
  - Sequência de comandos
  - Condicionais
  - Laços de repetição
- ✅ Interface gráfica interativa
- ✅ Feedback visual em tempo real

---

## 🛠️ Tecnologias utilizadas

- Python
- Pygame
- Estrutura modular para organização do código
